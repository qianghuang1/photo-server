# Photo Server

A small Node.js CLI that hosts a local web gallery for images under a folder.

## Usage

```sh
npm install
npx photo-server .
```

Options:

```sh
photo-server [folder] --host 127.0.0.1 --port 3000
```

Open the printed URL in a browser to browse folders, review photos in ratio-preserving thumbnail tiles, open original images with PhotoSwipe, multi-select images, and download the selected files individually.
